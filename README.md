# proxychains-gui
proxychains-gui is an application proxy manager written by ChatGPT's code. 

一个由 ChatGPT 编写代码得到的 QT 应用代理管理器，后端使用 proxychains 实现代理功能。

测试系统：Kylin Desktop Linux V10 SP1 2203

理论上支持 Debian/Ubuntu
